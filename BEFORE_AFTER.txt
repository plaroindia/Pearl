DARK MODE TRANSFORMATION
────────────────────────

LIGHT MODE (Default):
body {
    --primary: #2196f3
    --text: #1e293b
    --bg-light: #f8fafc
    background: #f8fafc
}

DARK MODE (When .dark class added):
body.dark {
    --primary: #60a5fa
    --text: #f1f5f9
    --bg-dark: #0f172a
    background: #0f172a
}

RESULT: ALL elements using var(--primary), var(--text), etc. automatically 
adapt without needing any JavaScript logic or manual updates!

MAINTENANCE ADVANTAGES
──────────────────────

OLD APPROACH (Hard-coded colors):
To change primary color from #667eea to #2196f3:
- Find all instances of #667eea
- Replace in 50+ places
- Risk of missing some
- Inconsistencies
- Time consuming

NEW APPROACH (CSS Variables):
To change primary color:
:root {
    --primary: #2196f3;
    /* All 100+ components update automatically */
}

Result: Centralized, consistent, easy to maintain
