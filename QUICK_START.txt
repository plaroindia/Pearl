API INTEGRATION
================================================================================

Frontend will use:
  Base URL: http://localhost:8000
  
Example API Call:
```typescript
const token = await getAuthToken(); // From Supabase auth
const response = await fetch('http://localhost:8000/api/skill-gap', {
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  }
});
const data = await response.json();
```

TROUBLESHOOTING
================================================================================

âŒ Problem: npm install fails
âœ… Solution: 
   - Use Node.js 16+ (check: node -v)
   - Clear cache: npm cache clean --force
   - Delete node_modules: rm -r node_modules
   - Try again: npm install

âŒ Problem: Port 8000 already in use
âœ… Solution: 
   - Find process: netstat -ano | findstr :8000 (Windows)
   - Kill process: taskkill /PID <PID> /F
   - Or change port in main.py

âŒ Problem: API calls return 401
âœ… Solution:
   - Verify token in localStorage/sessionStorage
   - Check Supabase auth is working
   - Ensure Authorization header is set

âŒ Problem: CSS not loading
âœ… Solution:
   - Check index.css exists in pearl-agent/
   - Verify import in index.tsx
   - Check browser console for errors

âŒ Problem: Hot reload not working
