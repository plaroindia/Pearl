DESIGN DECISIONS RATIONALE

1. CSS Variables Over Hard-Coded Colors
   Reason: Single source of truth for theming
   Benefit: Easy dark mode, consistent palette

2. Preserved All JavaScript
   Reason: Safety - zero functional regression
   Benefit: All features work exactly as before

3. Modern CSS Grid/Flexbox
   Reason: Responsive by nature
   Benefit: Better mobile experience automatically

4. Poppins Font from Pearl/
   Reason: Match design system
   Benefit: Professional, modern appearance

5. Single CSS Block
   Reason: Organized and maintainable
   Benefit: No duplicate styles, cleaner code
